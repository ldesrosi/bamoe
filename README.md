# BAMOE deployment

This repo sets up BAM OE 9.1 with underlying dependencies:
1. Kafka
2. Postgresql
3. Keycloak
4. BAMOE Services (data-index, management console, task console)
and
5. BAMOE Application

To install:  
  
`cd bamoe-app` 
`./deploy.sh` 

To reset the environment:  
  
`./cleanup.sh`
